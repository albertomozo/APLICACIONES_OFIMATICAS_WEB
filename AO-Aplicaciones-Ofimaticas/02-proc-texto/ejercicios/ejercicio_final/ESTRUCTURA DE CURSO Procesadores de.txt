ESTRUCTURA DE CURSO: Procesadores de Texto mediante Guías Rápidas

Objetivo general

El objetivo general de este curso es capacitar al alumnado para crear, formatear y presentar documentos profesionales usando un procesador de texto. Se busca que la persona se familiarice con las herramientas básicas y avanzadas que permiten elaborar documentos claros, organizados y visualmente coherentes. Al finalizar, el alumnado será capaz de aplicar las funciones esenciales de programas como Word, Writer o Google Docs para resolver situaciones reales.

Organización general

La duración sugerida del curso es de entre 20 y 30 horas, tiempo pensado para introducir conceptos, practicar con ejemplos guiados y realizar un proyecto final aplicado. La metodología se basa en guías rápidas que explican cada herramienta de forma breve y clara, combinadas con ejercicios prácticos que consolidan lo aprendido. El formato habitual es trabajar una guía por sesión o por cada bloque temático, de manera que el alumnado pueda avanzar de forma progresiva.

La evaluación consistirá en pequeños ejercicios aplicados y en la realización de un documento final completo. Este documento debe mostrar el dominio de las distintas herramientas vistas durante el curso, desde el formato básico del texto hasta la creación de tablas, imágenes o estilos automatizados.

Estructura tipo de cada guía

Cada guía mantiene el mismo formato para facilitar el aprendizaje y la comprensión. Todas incluyen un título del tema, un objetivo específico y una explicación de los conceptos clave, indicando qué es cada herramienta y para qué sirve. También presentan un apartado de pasos esenciales u opciones principales, acompañado de una breve comparativa entre Word, Writer y Google Docs para que el alumnado pueda identificar similitudes y diferencias.

Además, las guías incluyen consejos y buenas prácticas que ayudan a aplicar correctamente la herramienta en situaciones reales. Para consolidar la comprensión, cada guía finaliza con un ejercicio rápido y un ejercicio aplicado o mini-proyecto, lo que permite practicar de forma inmediata lo aprendido.

INDICE POR MÓDULOS

Módulo 1: Introducción y entorno

Guía 1 – Conocer el procesador de textos

En esta guía se explica qué es un procesador de textos y por qué es la herramienta básica para crear cualquier documento profesional. Se abordan las funciones esenciales como escribir, editar, corregir y dar formato, así como las ventajas respecto al uso de una simple máquina de escribir o de editores muy básicos. El objetivo es que el alumnado reconozca el programa como algo más que un espacio donde escribir, entendiendo su potencia y su utilidad en distintos contextos.

También se describen las partes principales de la interfaz, como la cinta de opciones, las barras de herramientas, el área de trabajo y las distintas vistas disponibles. Se pretende que el alumnado sepa moverse con soltura, localizando rápidamente las funciones más comunes. La guía finaliza mostrando para qué sirve cada tipo de vista del documento y cuándo es recomendable utilizarla.

Ejercicio: crear y guardar un documento.

Guía 2 – Crear, guardar y organizar documentos

Esta guía profundiza en el proceso de crear nuevos documentos y en la importancia de una correcta gestión de archivos. Se explican las diferencias entre Guardar y Guardar como, ya que es habitual que el alumnado confunda ambos conceptos. También se introducen los distintos formatos de archivo, como .docx, .odt y .pdf, aclarando en qué situaciones es conveniente usar cada uno y cuál es su compatibilidad con otras plataformas.

Además, se tratan temas como la recuperación de versiones anteriores, la importancia de guardar con frecuencia y la organización de documentos en carpetas y ubicaciones de almacenamiento. Se anima al alumnado a adoptar hábitos de orden digital que les faciliten encontrar y gestionar sus documentos en el futuro.

Módulo 2: Formato básico

Guía 3 – Dar formato al texto

Esta guía enseña a modificar la apariencia del texto mediante el uso de fuentes, tamaños, colores y estilos. Se explican las diferencias entre una fuente serif y una fuente sans serif, así como su impacto en la legibilidad. También se describe cómo aplicar estilos de negrita, cursiva y subrayado, y cuándo es apropiado utilizar cada uno para no sobrecargar el documento.

Otro aspecto importante es la alineación del texto: izquierda, centrado, derecha y justificado. Se analiza cómo influyen en la presentación del contenido y qué alineación suele usarse para cada tipo de documento. El alumnado comprenderá que el formato del texto es clave para transmitir profesionalidad.

Ejercicio: hoja informativa con diferentes estilos.

Guía 4 – Párrafos, interlineado y sangrías

En esta guía se profundiza en el formato del párrafo, uno de los elementos más importantes para organizar un documento. Se explica cómo ajustar el espaciado antes y después de un párrafo, así como las distintas opciones de interlineado. Con ejemplos prácticos se muestra cómo una correcta separación del texto mejora la legibilidad y la estructura.

También se introducen las sangrías y las tabulaciones, además del uso adecuado de listas con viñetas y listas numeradas. Se resalta que una buena organización del contenido facilita la comprensión y ayuda a presentar ideas de forma clara y ordenada.

Ejercicio: redactar un informe con párrafos y listas.

Módulo 3: Estructura de documento

Guía 5 – Opciones de página

En esta guía se detallan las opciones básicas de configuración de página, como los márgenes, la orientación y el tamaño del papel. Se explica cómo estas características afectan al diseño general del documento y por qué es importante definirlas antes de comenzar a maquetar contenido. También se introducen los encabezados y pies de página como elementos esenciales para documentos largos.

Además, se enseñan herramientas para insertar numeración de página y crear una portada separada sin numeración. El alumnado aprende a utilizar estas opciones para mejorar la presentación y estructura del documento desde el inicio.

Ejercicio: documento con portada y numeración.

Guía 6 – Estilos y formatos automáticos

Esta guía explica qué son los estilos y cómo permiten aplicar formatos uniformes a todo el documento de forma rápida. Se enseñan los estilos predefinidos, cómo aplicarlos y cómo modificarlos según las necesidades del documento. También se muestra cómo crear estilos nuevos para personalizar títulos, subtítulos y otros elementos.

Otro punto clave es la generación de un índice automático. El alumnado entenderá que, al utilizar estilos de título correctamente, el programa puede crear y actualizar un índice de forma automática. Esta herramienta resulta fundamental para documentos largos o trabajos profesionales.

Ejercicio: informe con título y subtítulos numerados.

Módulo 4: Elementos gráficos y tablas

Guía 7 – Insertar imágenes y dar formato

Esta guía introduce el uso de imágenes dentro del documento y explica cómo insertarlas desde el ordenador o desde fuentes online. Se detalla cómo ajustar su tamaño, recortarlas y aplicar efectos simples. También se estudia la posición de la imagen en relación con el texto, incluyendo opciones como alineación, desplazamiento y diferentes modos de ajuste.

Se destaca la importancia de elegir imágenes de buena calidad y respetar la coherencia visual del documento. Se anima al alumnado a evitar el abuso de decoraciones innecesarias para mantener la claridad y profesionalidad en sus trabajos.

Ejercicio: cartel informativo.

Guía 8 – Tablas, gráficos y listas avanzadas

En esta guía se explica cómo crear tablas y modificar su estructura mediante la combinación de celdas, la inserción de filas y columnas y la aplicación de estilos predefinidos. Se enseña a usar la tabla no solo como herramienta para organizar datos, sino también como elemento visual que ayuda a presentar información de forma clara.

Además, se introducen gráficos simples y las listas multinivel como herramientas avanzadas para estructurar información. El objetivo es que el alumnado pueda presentar datos de forma más visual y profesional.

Ejercicio: tabla comparativa o calendario.

Módulo 5: Automatización y diseño profesional

Guía 9 – Plantillas y formularios simples

Esta guía presenta las plantillas como una forma eficaz de reutilizar diseños y formatos ya establecidos. Se enseñan las plantillas predefinidas disponibles en los distintos procesadores de texto y cómo modificarlas para adaptarlas a necesidades específicas. También se explica cómo guardar una plantilla propia para utilizarla en futuros proyectos.

Además, se introducen los formularios simples mediante el uso de campos de texto, casillas y fechas. Se muestran ejemplos de situaciones reales donde estos formularios son útiles, como encuestas o documentos administrativos.

Guía 10 – Revisión, comentarios y control de cambios

En esta guía se aborda el proceso de revisión de un documento. Se explican las herramientas automáticas como el corrector ortográfico y gramatical, así como la importancia de revisarlas con criterio. También se enseña a insertar comentarios para colaborar con otras personas y facilitar la comunicación dentro del documento.

El control de cambios se presenta como una herramienta para registrar todas las modificaciones realizadas. Esto es especialmente útil en trabajos colaborativos, ya que permite aceptar o rechazar los cambios de manera clara y ordenada.

Guía 11 – Preparar para impresión y exportar a PDF

Esta guía explica las opciones de impresión del documento, incluyendo la configuración del diseño, la orientación, los márgenes y la vista previa. Se enseña cómo generar un archivo PDF para compartir o imprimir, garantizando que la apariencia final sea exactamente la deseada. También se explica cómo numerar secciones y cómo dejar la portada sin numeración visible.

Módulo 6: Proyecto final

Guía 12 – Proyecto: Documento completo profesional

En esta guía se recoge el proyecto final del curso, donde el alumnado debe crear un documento completo aplicando todas las herramientas vistas. Se propone elaborar un informe, una memoria o un folleto con formato profesional, incluyendo portada, índice, encabezados, pies de página, numeración, tablas, imágenes y estilos.

El objetivo es que el alumnado demuestre que sabe combinar todas las funciones del procesador de textos y que es capaz de presentar un documento bien organizado y visualmente coherente.
