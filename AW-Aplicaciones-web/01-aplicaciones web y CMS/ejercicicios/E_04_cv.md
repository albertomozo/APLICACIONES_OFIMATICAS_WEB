# **EJERCICIO FINAL: Crear y Publicar tu Currículum Online en Netlify**

## **Objetivo del ejercicio**

Partiendo del esqueleto de página web HTML proporcionado, deberás **crear tu propio Currículum Online** aplicando contenidos básicos de HTML y CSS.
El resultado final será una página web funcional que deberás **subir y publicar en Netlify**, para obtener una URL pública donde se pueda visualizar tu CV desde cualquier dispositivo.

---

# **Instrucciones del ejercicio**

## **1. Descarga o copia el esqueleto HTML**

Se te proporciona el siguiente archivo base:

**`index.html`**
(Es el esqueleto que incluye cabecera, menú, secciones, tablas, iconos SVG y estructura básica)

Tu tarea será personalizarlo y completarlo.

---

## **2. Personaliza los contenidos del CV**

Rellena todas las secciones con tus datos reales o ficticios:

### **Datos personales**

* Nombre y apellidos
* Email
* Teléfono (si se desea)
* Ciudad o país
* Enlace a LinkedIn (si existe)

### **Experiencia**

En cada artículo:

* Título del puesto
* Fechas
* Descripción breve
* Lista de funciones

### **Formación**

Incluye:

* Título del curso
* Nº de horas o nivel
* Centro
* Breve temario o competencias adquiridas

### **Idiomas**

Rellena la tabla indicando niveles y competencias de cada idioma.

### **Habilidades**

En cada tarjeta:

* Nombre de la habilidad (HTML, CSS, Word, Excel, etc.)
* Nivel (1 a 10)
* (Opcional) Añade un icono SVG o imagen pequeña

---

## **3. Estiliza la página con CSS**

En el bloque `<style>`, añade mejoras básicas como:

* Fuente legible (por ejemplo `Arial` o `Verdana`)
* Colores para la cabecera y el pie
* Líneas que separen secciones
* Estilo para el menú
* Espaciado entre elementos
* Ancho máximo del contenido (por ejemplo 900 px centrado)

*(Se permite crear un archivo CSS externo)*

---

## **4. Añade al menos 3 mejoras opcionales**

Elige **tres o más** de las siguientes:

* Añadir una foto tuya o avatar
* Colorear las filas de la tabla de idiomas
* Quitar viñetas y dar estilo a las listas
* Convertir el menú en un menú fijo (“sticky”)
* Cambiar el color del fondo de la web
* Añadir iconos SVG a más habilidades
* Añadir enlaces a redes sociales
* Crear una segunda página (por ejemplo, “Proyectos.html”)
* Añadir animaciones simples con CSS (transiciones al pasar el ratón, etc.)

---

## **5. Prepara el proyecto para subirlo a Netlify**

1. Crea una carpeta llamada **mi-cv-online**.
2. Dentro coloca:

   * `index.html`
   * (Opcional) tu carpeta `css/`
   * (Opcional) carpeta `img/`
3. Verifica que **al abrir index.html en el navegador**, todo funciona correctamente.

---

# **6. Publicación en Netlify**

### Opción A: Arrastrar carpeta (más sencilla)

1. Entra en [https://app.netlify.com/](https://app.netlify.com/)
2. Crea una cuenta o inicia sesión.
3. Haz clic en **"Add new site" > "Deploy manually"**.
4. Arrastra la carpeta **mi-cv-online** dentro del área indicada.
5. Espera la publicación.
6. Netlify te dará una URL del tipo
   `https://nombre-random.netlify.app`
7. Cópiala y entrégala en el campus o documento solicitado.

### Opción B: Subir desde GitHub (si sabes usar repositorios)

Opcional si el grupo ya trabaja con Git.

---

# **7. Entregables del ejercicio**

Debes entregar:

1. **Enlace público de tu CV** en Netlify
2. El archivo comprimido `.zip` del proyecto completo
3. Una frase final indicando:

   * Qué parte te ha resultado más difícil
   * Qué te gustaría mejorar en una siguiente versión
